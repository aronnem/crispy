IFS.reduceIFSMap(par, IFSimageName, method='optext', ivar=False)[source]¶

Main reduction function

Uses various routines to extract an IFS detector map into a spectral-spatial cube.

Parameters:

par : Parameter instance

Contains all IFS parameters

IFSimageName : string

Path of image file

method : ‘simple’, ‘dense’, ‘apphot’, ‘test’, ‘lstsq’, ‘optext’

Method used for reduction. ‘simple’: brute force photometry, adds up the fluxes in a line of pixels centered where the centroid at that wavelength falls. Not very accurate. ‘dense’: does the same but after interpolating/densifying the image by a factor of 10, which gives the previous method more accuracy (but depends on the interpolation scheme). ‘test’: use the method defined in the testReduction function, for experimenting. ‘lstsq’: use the knowledge of the PSFs at each location and each wavelength and fits the microspectrum as a weighted sum of these PSFs in the least-square sense. Can weigh the data by its variance. ‘optext’: use a matched filter to appropriately weigh each pixel and assign the fluxes, making use of the inverse wavlength calibration map. Then remap each microspectrum onto the desired wavelengths

ivar : Boolean

Uses the variance information. If the original image doesn’t have a variance HDU, then use the image itself as its own variance (Poisson noise). Default False.
